About Us

Supermarket Navigation is a one-stop supermarket simulation that aims to offer users a wide range of basic home and personal products under one roof. Users can select items from the store stocks home utility products - including food, garments, electronic goods, home appliances and more - available at competitive prices that our users appreciate. Our core objective is to offer users a new shopping experience at a great value saving time like never seen before.
Our Team has created the most flexible and scalable admin panel which provides the store to add various products with an ease. The items are then stored inside our firebase storage that provide fucntionality such as Realtime data projection, Admin Authentication and solid infrastructure security. The data stored is the parsed to the supermarket Navigation App provided to our users using which they can select all the desired items from the "Store Tab" inside the App, which then based on the count of items present inside the MyList Tab the users will be guided through the store taking the shortest path available provided by our very own Alogrithm.
